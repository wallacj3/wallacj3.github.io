Concentrix Pojects


Main Landing Page for Concentrix CORE

This application was built within SharePoint 2016. The initial requirements were to keep the functionality of SharePoint while having a purely HTML home page. The search bar was modified from the original SharePoint version using CSS edits. The carousel on the left and hover effects on the right we're built using Bootstrap css and pure JavaScript.


Community Pages for Concentrix Department Groups

This application was built within SharePoint 2016. These pages were designed with the intention of giving various groups throughout the corporation a place to house their process documentation and post announcements to their team members. This page utilizes the Modal popup function in Bootstrap as well as the "popover" function to give users additional information about each section.


Home Pages for site level employees


Korean Version

These landing pages where built as a way to connect site level employees to the knowledge content within CORE. The challenge we faced was creating and directing users to appropriate pages for their country of origin. I was able to achieve this with a combination of Javascript and native functionality within SharePoint. In addition to presenting country specific content, I was tasked with creating a way to push global announcements to all pages but still have the local annoucements appear to only the users of that country. I achieved this effect through Javascript and CSS classes.
